# 🐾 Digital Pet Pro

Digital Pet Pro is a Tamagotchi-style mobile application developed using **React Native and Expo**.  
The application allows users to interact with a virtual pet and take care of it through various gamified actions such as feeding, playing, and monitoring its status.

This project was developed as part of a **mobile application development assignment** focusing on gamification and mobile development concepts.

---

# 🎮 Gamification Features

The application includes several gamification elements:

- 🐶 **Virtual Pet System** – Users interact with a digital pet.
- 🍔 **Feeding Mechanic** – The user feeds the pet to maintain energy.
- 🎮 **Interaction Actions** – Play and interact with the pet.
- 😊 **Mood & Energy System** – The pet has happiness and energy levels.
- 🏆 **Gamified Experience** – Regular interaction keeps the pet healthy.

These mechanics simulate the classic **Tamagotchi-style virtual pet experience**.
